## Mock Interview Solution Format

```
Start time: [HH:MM AM/PM]

[INTERVIEW PROBLEM PROMPT]
[Examples provided in problem]
[Constraints]

Answer:
[Your understanding of the problem and what it's asking for]

### Observations:
[Key insights about the problem structure, patterns, or relationships]

### Edge Cases:
[List of edge cases to handle - BE THOROUGH, USE CHECKLIST]

### Potential Solutions:
1. **[Approach Name]** - O(x) time, O(x) space
   [Brief description]
2. **[Approach Name]** - O(x) time, O(x) space
   [Brief description]

### Implementation Decision:
I will implement [chosen approach] because [clear reasoning for choice, including trade-offs]

[IF RECURSIVE: State recursion safety checklist here]

### Code Implementation:
[Your solution code]

### Code Review (30-Second Scan):
[Checklist items verified before submission]

### Code Trace:
[Step-by-step walkthrough of your code with a sample input]

1st submission time: [HH:MM AM/PM]

[If applicable:]
2nd submission time: [HH:MM AM/PM]
[Changes made]

Follow-up Questions: (Time: [HH:MM AM/PM])
[Your responses to follow-up questions]

End time: [HH:MM AM/PM]
```
